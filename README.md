# dogtor-app

Antes de programar, você vai precisar installar as dependencias do projeto!
comando:
```
npm install --save-dev
```

a flag "--save-dev" indica que também serão instaladas as dependenciais de desenvolvimento
como o typescript (compilador)


Talvez sejá necessário executar o comando
```
npm install typescript -g
```

a flag -g indica que o typescript será instalado globalmente, ou seja, você poderá executá-lo mesmo fora de projetos typescript
